۵بخش‌های ناقص

P0 — بحرانی

یکپارچه‌سازی با VPN Bot Pro API

اپ از Cloudflare Worker استفاده نمی‌کند. باید endpoint های /api/configs، /api/countries، /api/sub مستقیماً فراخوانی شوند.

P0 — بحرانی

سیستم Vote / Auto-Like

بعد از هر تست TCP موفق، باید POST /dashboard/api/vote با "vote":"like" ارسال شود تا Quality Score بهبود یابد.

P0 — بحرانی

سیستم Subscription Code

کاربران باید کد اشتراک (AdminID-ClientID) وارد کنند و از GET /api/user-sub?code=... کانفیگ شخصی دریافت کنند.

P1 — مهم

Country Filter با UI Picker

فراخوانی GET /api/countries و نمایش BottomSheet/Modal برای انتخاب کشور با پرچم emoji.

P1 — مهم

کپی Raw Config + QR Code

نمایش vless://... در فرمت monospace، دکمه کپی با Haptic feedback، و QR Code برای import.

P1 — مهم

بررسی آپدیت اپ + اعلانات

فراخوانی /api/app-update هنگام راه‌اندازی و نمایش Modal اجباری اگر force: true. نمایش Banner اعلانات از /api/announcements.

P2 — توسعه

جستجو + مرتب‌سازی + صفحه جزئیات

SearchBar با فیلتر real-time، مرتب‌سازی چندگانه، و Detail View با QR Code و share.
